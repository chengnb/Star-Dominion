# 星域争霸 · Star Dominion

一个由 Claude Code + DeepSeek 全程编写的网页游戏。

多人实时太空策略网页游戏。指挥舰队，征服星系，探索无限宇宙。

## 快速开始

**前置条件**: PostgreSQL 运行在 `localhost:5432`，用户 `postgres`，密码 `Aa123456`

```powershell
cd server
npm install
node index.js
```

浏览器打开 `http://localhost:3000`，注册账号即可开始。

## 技术栈

| 层 | 技术 | 说明 |
|---|------|------|
| 服务器 | Node.js + Express | HTTP 静态服务 |
| 实时通信 | Socket.IO (WebSocket) | 20 TPS 状态同步, volatile 模式抗网络抖动 |
| 认证 | bcrypt + JWT | 密码哈希, 7 天令牌有效期 |
| 存储 | PostgreSQL | 支持并发读写，行星和玩家状态持久化 |
| 前端 | HTML5 Canvas | 原生渲染, 无框架依赖 |
| 空间索引 | 网格分区 (Map 实现) | 500px 单元格, O(1) 视口查询, 无坐标上限 |

## 游戏机制

### 无限世界

世界按 2000×2000 区块划分，使用确定性种子 PRNG 生成行星。玩家探索边界时自动生成新区块并持久化到 PostgreSQL。行星位置固定，行星名称使用唯一编号（行星-1, 行星-2 ...）。

### 战争迷雾

玩家只能看到己方行星和己方舰队周围 700 单位范围内的行星。随着扩张和舰队航行逐步揭开地图。己方行星列表随时可快捷跳转。

### 行星

| 类型 | 矿物/秒 | 能量/秒 | 颜色 |
|------|---------|---------|------|
| 岩石行星 | 3 | 1 | 棕色 |
| 气态巨行星 | 1 | 3 | 金色 |
| 类地行星 | 2 | 2 | 绿色 |

### 飞船

| 飞船 | 造价 (矿/能) | 速度 | HP | 伤害 |
|------|-------------|------|-----|------|
| 侦察机 | 50/30 | 180 | 30 | 5 |
| 战斗机 | 100/50 | 130 | 80 | 15 |
| 战列舰 | 250/150 | 80 | 250 | 40 |

### 战斗

- 舰队到达无主行星时直接占领
- 舰队到达敌方/离线玩家行星时自动交战 — 进攻方 vs (防御工事 + 驻军)
- 敌对舰队在途中相遇也会交战 (30 单位触发距离)
- 行星可升级防御等级, 每级 +100 HP / +20 伤害
- 离线玩家的行星保留防御和驻军，仍需要正常战斗才能占领

### 持久化

玩家断开连接后行星保留所有权、防御等级和驻军。重连时自动恢复所有行星和资源。行星全被占领时自动分配新母星。

## 操作

| 操作 | 效果 |
|------|------|
| 左键拖动 | 平移视角 |
| 滚轮 | 缩放 (0.25x ~ 3x) |

## 架构要点

- **权威服务器**: 所有游戏逻辑在服务端执行, 客户端仅渲染和输入
- **战争迷雾**: `_computeFogVisibility` 基于己方行星/舰队计算可见行星集合
- **双重脏标记**: `dirtyPlanets` 每 tick 清空用于网络同步, `_pendingSave` 仅保存后清空用于持久化
- **区块生成**: 惰性加载, 种子化确定生成, 探索时自动扩展
- **用户缓存**: 离线玩家行星仍显示拥有者名称和颜色
- **舰队航线**: 虚线显示舰队从当前位置到目的地的路径
- **快捷面板**: 左侧己方行星列表，文本增量更新避免闪烁
