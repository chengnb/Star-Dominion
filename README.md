# 星域争霸 · Star Dominion
一个由claude code+deepseekv4全程编写的网页游戏
多人实时太空策略网页游戏。指挥舰队, 征服星系, 与数百玩家争夺宇宙霸权。

## 快速开始

```powershell
cd server
npm install
node index.js
```

浏览器打开 `http://localhost:3000`, 注册账号即可开始。

## 项目结构

```
├── server/                  # 后端 — 权威游戏服务器
│   ├── index.js             # 入口: Express + Socket.IO + 游戏主循环
│   ├── db.js                # JSON 文件数据库 (用户账户 & 游戏状态持久化)
│   ├── game/
│   │   ├── constants.js     # 游戏常量 (地图尺寸、行星类型、飞船参数)
│   │   ├── SpatialGrid.js   # 空间分区网格 — 高效视口裁剪查询
│   │   └── GameWorld.js     # 游戏世界核心 — 所有游戏逻辑的权威实现
│   └── package.json
├── client/                  # 前端 — Canvas 渲染 + UI
│   ├── index.html           # 主页面 (登录界面 + 游戏画布)
│   ├── css/style.css        # 样式
│   └── js/
│       ├── main.js          # Socket.IO 连接 & 认证流程
│       └── Game.js          # 渲染引擎 / 镜头控制 / 输入 / HUD / 小地图
└── data/
    └── gamedata.json        # 持久化数据 (自动生成)
```

## 技术栈

| 层 | 技术 | 说明 |
|---|------|------|
| 服务器 | Node.js + Express | HTTP 静态服务 & API |
| 实时通信 | Socket.IO (WebSocket) | 20 TPS 状态同步, volatile 模式抗网络抖动 |
| 认证 | bcrypt + JWT | 密码哈希, 7 天令牌有效期 |
| 存储 | JSON 文件 | 零依赖, 每 30 秒自动保存 |
| 前端 | HTML5 Canvas | 原生渲染, 无框架依赖 |
| 空间索引 | 网格分区 | 500px 单元格, O(1) 视口查询 |

## 游戏机制

### 行星

地图 5000×5000, 随机分布 150 颗行星, 分三种类型:

| 类型 | 矿物/秒 | 能量/秒 | 颜色 |
|------|---------|---------|------|
| 岩石行星 | 3 | 1 | 棕色 |
| 气态巨行星 | 1 | 3 | 金色 |
| 类地行星 | 2 | 2 | 绿色 |

### 飞船

| 飞船 | 造价 (矿/能) | 速度 | HP | 伤害 |
|------|-------------|------|-----|------|
| 侦察机 | 50/30 | 180 | 30 | 5 |
| 战斗机 | 100/50 | 130 | 80 | 15 |
| 战列舰 | 250/150 | 80 | 250 | 40 |

### 战斗

- 舰队到达敌方行星时自动交战 — 综合攻击力高者获胜
- 敌对舰队在途中相遇也会交战 (30 单位触发距离)
- 行星可升级防御等级, 每级 +100 HP / +20 伤害

## 操作

| 操作 | 效果 |
|------|------|
| 左键拖动 | 平移视角 |
| 滚轮 | 缩放 (0.25x ~ 3x) |
| 左键点行星 | 选中, 显示驻军 & 建造面板 |
| 右键点行星 | 从选中行星派遣全部驻军到目标 |
| H | 回到母星 |
| Esc | 取消选中 |
| 回车 | 聊天 (焦点需在聊天输入框) |

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `PORT` | `3000` | 服务器端口 |
| `JWT_SECRET` | 内置默认值 | JWT 签名密钥 (生产环境务必修改) |

## 架构要点

- **权威服务器**: 所有游戏逻辑 (建造、派遣、战斗、资源) 在服务端执行, 客户端仅作渲染和输入
- **视口裁剪**: 每个玩家只收到视野范围内的实体更新, 降低带宽
- **脏标记**: 发生变更的行星 (易主/战斗) 无视视口广播, 保证小地图实时准确
- **平滑镜头**: 客户端镜头缩放使用指数衰减插值
